## PART I: 数据库基础操作代码

### 1.1 数据导入（使用Access向导，非代码）
**手动步骤**：
1. 打开Access，创建新数据库
2. 外部数据 → 文本文件 → 选择TXT文件
3. 关键设置：
   - 表名：`TCM_Prescription_Table`
   - `Physician_ID`字段 → 文本类型

   - 勾选"不重复记录"

## PART II: 查询设计代码

### 2.1 查询1：医生处方统计
**查询名称**: `1_Count_Physician_Prescriptions_and_Amounts`

```sql
SELECT 
    d.Physician_Name AS 医生姓名,
    COUNT(p.Prescription_ID) AS 处方数量,
    SUM(p.Prescription_Amount) AS 处方小计
FROM 
    Doctor_Table d
    INNER JOIN TCM_Prescription_Table p 
    ON d.Physician_ID = p.Physician_ID
WHERE 
    LEN(d.Physician_Name) = 3
    AND d.Physician_Name NOT LIKE '%[0-9]%'
    AND d.Physician_Name NOT LIKE '%[a-zA-Z]%'
GROUP BY 
    d.Physician_Name;
```

### 2.2 查询2：年龄参数查询
**查询名称**: `2_Query_Patients_by_Age_Range`

```sql
PARAMETERS 
    [请输入最小年龄。] Long,
    [请输入最大年龄。] Long;
SELECT 
    Patient_Name AS 患者姓名,
    Patient_Gender AS 性别,
    Patient_Age AS 年龄,
    Clinical_Diagnosis AS 临床诊断
INTO Patients_Aged_25_to_40
FROM TCM_Prescription_Table
WHERE Patient_Age BETWEEN [请输入最小年龄。] AND [请输入最大年龄。];
```

### 2.3 查询3：处方金额更新
**查询名称**: `3_Increase_Prescription_Amounts`

```sql
UPDATE TCM_Prescription_Table
SET Prescription_Amount = Prescription_Amount * 1.05
WHERE Prescription_Amount < 150;
```

### 2.4 查询4：交叉表统计
**查询名称**: `4_Count_Patients_by_Department_and_Physician`

```sql
TRANSFORM 
    Count(TCM_Prescription_Table.Patient_Name) AS 患者计数
SELECT 
    Department_Table.Department_Name AS 科室名称,
    Doctor_Table.Physician_Name AS 医生姓名
FROM 
    (Department_Table 
    INNER JOIN Doctor_Table 
    ON Department_Table.Department_ID = Doctor_Table.Department_ID) 
    INNER JOIN TCM_Prescription_Table 
    ON Doctor_Table.Physician_ID = TCM_Prescription_Table.Physician_ID
GROUP BY 
    Department_Table.Department_Name,
    Doctor_Table.Physician_Name
PIVOT 
    TCM_Prescription_Table.Patient_Gender;
```

```vba
' 报表的打开事件
Private Sub Report_Open(Cancel As Integer)
    If IsNull([Forms]![frmBrowsePrintPrescription]![cboPhysician]) Then
        MsgBox "请先从表单中选择医生", vbInformation
        Cancel = True
    Else
        Me.Filter = "Physician_ID = '" & [Forms]![frmBrowsePrintPrescription]![cboPhysician] & "'"
        Me.FilterOn = True
    End If
End Sub
```

## 故障排除

### 常见问题1：SQL执行错误
**问题**：执行ALTER TABLE时出错
**解决**：在Access设计视图中手动修改字段类型

### 常见问题2：查询2参数不提示
**问题**：运行查询2时没有提示输入年龄
**解决**：确保使用PARAMETERS语句，或在查询设计视图中设置参数

### 常见问题3：表单按钮不工作
**问题**：点击按钮没有反应
**解决**：检查按钮事件是否绑定到正确的VBA过程

### 常见问题4：报表不显示数据
**问题**：报表打开时空白
**解决**：检查筛选条件，确保从表单传递了医生ID

### 常见问题5：关系建立失败
**问题**：无法建立级联关系
**解决**：确保两个表的Physician_ID字段类型完全一致
