## 🛠️ 分步操作指南

### 第一步：打开Access，创建数据库（15分钟）
1. 打开 **Microsoft Access 2016或更高版本**
2. 点击 **文件 → 新建 → 空白数据库**
3. 文件名输入：`TCM Prescription Management Database.accdb`
4. 点击 **创建**

### 第二步：导入TXT数据（30分钟）
1. 点击顶部菜单 **外部数据 → 文本文件**
2. 找到并选择 `TCM_Prescription_Information.txt`
3. 点击 **确定**，进入导入向导
4. **关键设置**：
   - 选择"带分隔符"
   - 第一行包含字段名：**勾选**
   - 导入到新表：**勾选**
   - 表名：`TCM_Prescription_Table`
5. 在字段选项步骤：
   - 找到 `Physician_ID` 字段 → 数据类型选择 **文本**
   - 找到 `Patient_Gender` 字段 → 数据类型选择 **是/否**
6. 完成导入

### 第三步：修改表结构（30分钟）
1. 在左侧导航栏，右键 `TCM_Prescription_Table` → **设计视图**
2. 按以下要求修改字段：
   ```
   字段名            数据类型    字段大小
   Prescription_ID   文本        10
   Patient_Name      文本        10  
   Patient_Gender    是/否       -
   Patient_Age       数字        整型
   Clinical_Diagnosis 文本       50
   Formula_Name      文本        30
   Formula_Composition 备注      -
   Physician_ID      文本        10
   Prescription_Date 日期/时间   -
   Prescription_Amount 货币      -
   ```
3. 设置主键：右键 `Prescription_ID` 字段 → **主键**
4. 创建索引：在 `Formula_Name` 字段属性中 → 索引：**有（有重复）**
5. 保存并关闭

### 第四步：建立表关系（20分钟）
1. 点击顶部菜单 **数据库工具 → 关系**
2. 添加 `Doctor_Table` 和 `TCM_Prescription_Table`
3. 从 `Doctor_Table` 拖拽 `Physician_ID` 到 `TCM_Prescription_Table` 的 `Physician_ID`
4. 在编辑关系对话框中：
   - ☑ 实施参照完整性
   - ☑ 级联更新相关字段
   - ☑ 级联删除相关记录
5. 点击 **创建**，保存关系

### 第五步：插入学生记录（10分钟）
1. 双击打开 `Doctor_Table`
2. 在最后一行添加新记录：
   ```
   Department_ID: K1
   Physician_ID: [你的学号后6位]
   Physician_Name: [你的姓名]
   Physician_Gender: [你的性别]
   Physician_Age: [你的年龄]
   Physician_Title: [你的职称，如"学生"]
   Physician_Education: [你的学历，如"本科"]
   Photo: (留空)
   ```
3. 保存

## 🔍 查询设计简化版

### 查询1：医生处方统计（20分钟）
1. 创建 → 查询设计
2. 添加 `Doctor_Table` 和 `TCM_Prescription_Table`
3. 添加字段：`Physician_Name`、`Prescription_ID`、`Prescription_Amount`
4. 在 `Physician_Name` 条件中输入：`Len([Physician_Name])=3`
5. 点击 **汇总** 按钮（Σ）
6. `Prescription_ID` → 总计：计数
7. `Prescription_Amount` → 总计：合计
8. 保存为：`1_Count_Physician_Prescriptions_and_Amounts`

### 查询2：年龄参数查询（20分钟）
1. 创建 → 查询设计
2. 添加 `TCM_Prescription_Table`
3. 添加字段：`Patient_Name`、`Patient_Gender`、`Patient_Age`、`Clinical_Diagnosis`
4. 在 `Patient_Age` 条件中输入：`Between [请输入最小年龄。] And [请输入最大年龄。]`
5. 点击 **生成表** 按钮
6. 表名输入：`Patients_Aged_25_to_40`
7. 保存为：`2_Query_Patients_by_Age_Range`

### 查询3：金额更新查询（15分钟）
1. 创建 → 查询设计
2. 添加 `TCM_Prescription_Table`
3. 点击 **更新** 按钮
4. 添加字段：`Prescription_Amount`
5. 更新到：`[Prescription_Amount]*1.05`
6. 条件：`<150`
7. 保存为：`3_Increase_Prescription_Amounts`

### 查询4：交叉表查询（25分钟）
1. 创建 → 查询设计 → 交叉表查询
2. 添加 `Department_Table`、`Doctor_Table`、`TCM_Prescription_Table`
3. 行标题：`Department_Name`、`Physician_Name`
4. 列标题：`Patient_Gender`
5. 值：`Patient_Name`，总计：计数
6. 保存为：`4_Count_Patients_by_Department_and_Physician`

## 📱 表单设计简化版

### 表单1：医生信息管理（60分钟）
1. 创建 → 窗体向导
2. 选择 `Doctor_Table` 的所有字段
3. 布局：纵栏表
4. 标题：医生基本信息管理
5. 完成后，右键表单 → **设计视图**
6. **重要属性设置**：
   - 格式 → 弹出方式：是
   - 格式 → 边框样式：对话框
   - 格式 → 导航按钮：无
   - 格式 → 记录选择器：无
7. 添加命令按钮（使用按钮向导）：
   - 首条记录、上一条记录、下一条记录、末条记录、关闭表单
8. 保存为：`frmPhysicianBasicInfo`

### 表单2：处方浏览打印（45分钟）
1. 创建 → 窗体设计（空白窗体）
2. 属性 → 标题：浏览和打印处方信息
3. 添加组合框控件（使用向导）：
   - 选择"自行键入所需的值"（先简化）
   - 列数：2，输入医生ID和姓名
4. 添加子窗体控件：
   - 选择"使用现有的表和查询"
   - 选择 `TCM_Prescription_Table` 的5个字段
5. 添加按钮：
   - 刷新按钮：VBA代码 `Me.Requery`
   - 预览按钮：VBA代码 `DoCmd.OpenReport "rptPhysicianPrescriptionList", acViewPreview`
6. 保存为：`frmBrowsePrintPrescription`

## 📄 报表设计简化版（45分钟）
1. 创建 → 报表向导
2. 选择 `TCM_Prescription_Table` 的所有字段
3. 分组：按 `Physician_ID`
4. 保存为：`rptPhysicianPrescriptionList`
5. 设计视图调整：
   - 报表标题：添加标签 `=[Forms]![frmBrowsePrintPrescription]![Combo0] & "处方列表"`
   - 页脚：添加文本框 `="总计: " & Count(*) & "张处方"`
   - 页脚：添加文本框 `="累计处方金额: " & Sum([Prescription_Amount]) & "元"`

## 🆘 遇到问题怎么办？

### 常见问题解决方案
1. **导入失败**：检查TXT文件路径，确认文件存在
2. **查询错误**：先创建选择查询测试条件，再转为其他查询
3. **表单打不开**：检查控件数据源是否正确
4. **关系错误**：确保两个表的连接字段类型完全一致
5. **字体找不到**：用相似字体替代，如宋体替代隶书
